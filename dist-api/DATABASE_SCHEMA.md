## üìã Tabela: `public.profiles`

**Gerenciada por:** Aplica√ß√£o  
**Acesso:** Via `.from('profiles')`  
**Relacionamento:** `profiles.id = auth.users.id` (FK CASCADE)

### Schema SQL

```sql
CREATE TABLE public.profiles (
  -- üîë Identifica√ß√£o
  id UUID NOT NULL,                        -- FK para auth.users.id
  
  -- üë§ Dados Pessoais
  cpf VARCHAR(14) NOT NULL,                -- ‚Üê CPF EST√Å AQUI (formato: "70109948467")
  full_name VARCHAR(255) NOT NULL,         -- ‚Üê NOME EST√Å AQUI
  avatar_url TEXT NULL,
  
  -- üéÅ Sistema de Refer√™ncia
  referral_code VARCHAR(20) NOT NULL,      -- C√≥digo √∫nico do usu√°rio
  referred_by UUID NULL,                   -- Quem indicou este usu√°rio
  
  -- ‚úÖ Status
  email_verified BOOLEAN DEFAULT FALSE,
  
  -- üïê Timestamps
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  
  -- üîê Constraints
  CONSTRAINT profiles_pkey PRIMARY KEY (id),
  CONSTRAINT profiles_cpf_key UNIQUE (cpf),
  CONSTRAINT profiles_referral_code_key UNIQUE (referral_code),
  CONSTRAINT profiles_id_fkey FOREIGN KEY (id) 
    REFERENCES auth.users (id) ON DELETE CASCADE,
  CONSTRAINT profiles_referred_by_fkey FOREIGN KEY (referred_by) 
    REFERENCES auth.users (id)
) TABLESPACE pg_default;

-- üìë √çndices
CREATE INDEX idx_profiles_cpf ON public.profiles USING btree (cpf);
CREATE INDEX idx_profiles_referral_code ON public.profiles USING btree (referral_code);
CREATE INDEX idx_profiles_referred_by ON public.profiles USING btree (referred_by);

-- üîÑ Trigger
CREATE TRIGGER update_profiles_updated_at 
  BEFORE UPDATE ON profiles 
  FOR EACH ROW 
  EXECUTE FUNCTION update_updated_at_column();
```

### 1Ô∏è‚É£ Fun√ß√£o RPC para verificar email

```sql
CREATE OR REPLACE FUNCTION check_email_exists(email_to_check TEXT)
RETURNS BOOLEAN
LANGUAGE plpgsql
SECURITY DEFINER
AS $$
BEGIN
  RETURN EXISTS (
    SELECT 1 FROM auth.users WHERE email = email_to_check
  );
END;
$$;
```

Uso:
```javascript
const { data: exists } = await supabase.rpc('check_email_exists', { 
  email_to_check: email 
});
```

### 2Ô∏è‚É£ View para JOIN autom√°tico

```sql
CREATE OR REPLACE VIEW public.users_full AS
SELECT 
  u.id,
  u.email,
  u.email_confirmed_at,
  u.created_at as auth_created_at,
  p.cpf,
  p.full_name,
  p.avatar_url,
  p.referral_code,
  p.referred_by,
  p.email_verified,
  p.created_at as profile_created_at
FROM auth.users u
LEFT JOIN public.profiles p ON p.id = u.id;
```

Uso (com RLS apropriado):
```javascript

  .from('users_full')
  .select('*')
  .eq('id', userId)
  .single();
```
